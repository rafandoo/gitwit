# GitWit Commands

GitWit provides commands for installing and removing the tool from the repository, creating standardized commits, validating commit messages, and automatically generating changelogs.

In the following sections, you will find a description of each command, its available options and practical examples of use.

<br>

::: info 💡 Tip:
To view the help and all available options for any command, use:

```bash
gitwit <command> --help
```
:::
