# Comandos do GitWit

O GitWit disponibiliza comandos para instalação e remoção da ferramenta no repositório, criação de commits padronizados,
validação de mensagens de commit e geração automática de changelogs.

Nas seções seguintes, você encontrará a descrição de cada comando, suas opções disponíveis e exemplos práticos de
utilização.

<br>

::: info 💡 Dica:
Para visualizar a ajuda e todas as opções disponíveis de qualquer comando, utilize:

```bash
gitwit <comando> --help
```
:::
